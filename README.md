# FL-DEV

## 概要

Flower AI を使用したシュミレーション及び開発環境

## 環境構築

### 前提

- WSL, Docker Desktop, VSCodeがインストール済み

### 追加作業

- VSCode で Dev Containers 拡張機能をインストール

## 使用方法

1. Docker Desktop を起動
2. WSL 上にコードを展開
3. VSCode で開く
4. 「コンテナで再度開く」を選択してコンテナを起動
   - 必要なライブラリのインストールまで自動で行われる
   - インストールの終了まで待つ
